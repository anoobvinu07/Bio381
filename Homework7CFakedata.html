The data set generated will be to test the hypotesis that water use efficiency (WUE) for moist deciduous trees are unaffected by changes in temperature.

n <- 30 # sample size
height <- runif(n, min = 0.2, max = 1.3) # height increment (dependent)
WUE <- runif(n, min = -0.01, max = 0.99) # wateruse efficiency (dependent)
temp <- runif(n, min = 4, max = 26) # temperature in degree celsius (independent)
temp1 <- rep(c("hot","cold"), 15)
ID <- seq_len(n)
expData <- data.frame(ID,temp,height,WUE)

Though temperature seems to affect the WUE, it is not significant enough to reject the hypothesis.

Fixing the mean and standard deviation to a specific range

Replacing runif() with rnorm() for data generation
n <- 100 
height <- rnorm(n, mean = 0.5, sd = 0.25) 
WUE <- rnorm(n, mean=0.45,sd=0.20)
temp <- rnorm(n, mean=15,sd=0.10) 
ID <- seq_len(n)
expData <- data.frame(ID,temp,height,WUE)

When the sample size was set to 100 and the mean fixed for the variables (mean = 0.45, sd = 0.20 for WUE and mean = 15, sd = 0.10 for temperature), the p-value was not significant 95% of the times the experiment was run. Thus concluding that WUE is unaffected by changes in the temperature.
